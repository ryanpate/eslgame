## Color Scheme

- **Primary Orange**: #ff6b35 (pumpkin orange)
- **Secondary Orange**: #f7931e (bright orange)
- **Purple Accent**: #9b59b6 (spooky purple)
- **Dark Background**: #1a1a2e (dark blue-black)
- **Card Background**: #16213e (navy blue)
- **Success Green**: #2ecc71
- **Error Red**: #e74c3c

## Mobile View

The game automatically adjusts for mobile:
- Single column layout
- Larger touch targets
- Simplified navigation
- Full-screen cards
- Easy-to-read text

## Key Design Features

✅ **High Contrast** - Easy to read
✅ **Large Buttons** - Easy to click
✅ **Emoji Rich** - Visual learning
✅ **Responsive** - Works everywhere
✅ **Colorful** - Engaging for kids
✅ **Clear Feedback** - Instant responses
✅ **Progress Tracking** - Motivation
✅ **No Clutter** - Focus on learning

---

**The game is designed to be intuitive - students can start playing immediately without instructions!** 🎃
